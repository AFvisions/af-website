Project Details
This project is one that I inherited from my client’s former designer and because I didn’t have the native files I chose to completely recreate the digital brochure using InDesign so that current and future changes that the brochure needed could easily be accomplished. Highlights of the brochure include a spanish version and a printable version, high resolution stock photos, and within the “360 Method” page a 3D model I created in Blender.
